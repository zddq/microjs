# cron

crontab 语法 JavaScript 定时任务执行库

## cron 语法

```js
 # ┌────────────── second (可选)
 # │ ┌──────────── minute
 # │ │ ┌────────── hour
 # │ │ │ ┌──────── day of month
 # │ │ │ │ ┌────── month
 # │ │ │ │ │ ┌──── day of week
 # │ │ │ │ │ │
 # │ │ │ │ │ │
 # * * * * * *
```

## cron 值

| 字段   | 描述                   | 取值 | 特殊字符        |
| ------ | ---------------------- | ---- | --------------- |
| second | 秒数(可选)             | 0-59 | `, - * /`       |
| minute | 分钟                   | 0-59 | `, - * /`       |
| hour   | 小时                   | 0-23 | `, - * /`       |
| day    | 日期(日)               | 1-31 | `, - * / L W ?` |
| month  | 月份                   | 1-12 | `, - * /`       |
| week   | 星期几(0或7代表星期天) | 0-7  | `, - * / L ?`   |

## cron 特殊字符

- `*`：表示所有值
- `,`：指定多个值，例如 1,3,5 表示第 1、3、5 个值。
- `-`：指定范围，例如 1-5 表示第 1 到第 5 个值。
- `/`：指定步长，例如 \*/5 表示每 5 个值。
- `L`：表示最后一个，例如 L 表示每月的最后一天，5L 表示每月的最后一个星期五。
- `W`：表示最近的工作日。
- `?`：表示不指定，用于日期和星期字段。
